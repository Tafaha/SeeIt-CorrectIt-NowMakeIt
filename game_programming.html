Gaming Programming

Bachelor´s degree

About Gaming Programming:

Video game programmers develop code for games for web, console, PC, and mobile devices. Video game programming and specific game coders have wide-ranging applications in the entertainment, business, research, and training industries. Game Programming degree graduates master the coding and programming principles and techniques in all video game programming disciplines, in addition to C++ programming, scripting, data handling, DirectX/Open GL development, game engine architecture, game play interaction, artificial intelligence, networking and the use of middleware and industry APIs.
